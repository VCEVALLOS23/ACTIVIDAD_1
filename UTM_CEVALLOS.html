Crear una página HTML

Crear una página HTML sencilla con varios elementos, estos elementos deben tener su estilo modificado mediante un archivo CSS y mostrar el funcionamiento de las propiedades más elementales de modificación de estilos utilizando CSS..

Recomendaciones

  • Crear un documento HTML con varios elementos que puedan mostrar la modificación e importar el archivo CSS a crear.
  • Crear un archivo CSS con las propiedades más utilizadas y/o conocidas, aplicadas a un elemento utilizando cualquier tipo de selector para dicha tarea.
  • Empaquetar los dos archivos en un ZIP o un RAR para enviar la tarea.
